Increment

Provide the Increment attribute with an appropriate field you would wish to be incremented by one each time the button is pressed.

Im a Increment button 

<beer-button increment="number" id="inc">Im a Increment button</beer-button>

Decrement

Provide the Decrement attribute with an appropriate field you would wish to be decremented by one each time the button is pressed.

Im a Decrement button 

<beer-button decrement="decnumber" id="dec">Im a Decrement button</beer-button>

Navigate to a New Page

Provide the link attribute with a valid website link and it will navigate the browser to that page when the button is clicked.

I'm a link button 

<beer-button link="./testLink.html" functionName="testFunction"id="bill">Im a link button</beer-button>

Custom Function

By adding a JavaScript path to the script attribute and a function name to the functionName attribute, the user can create their own function and use it by clicking on the button.

I'm a Custom Function button 

<beer-button script="./testFunction.js" functionName="testFunction"id="func">Im a Custom Function button</beer-button>

Add your own image to a button

Provide a string corresponding to the path to an image. This will set the background of your button to the image.

At the moment, it doesn't resize the image or the button, it just puts the image in the button as it can.

Image Testing 

<beer-button image="test.jpg"id="img-test">Image Testing</beer-button>

Disabled Button

Provide the Disable attribute to disable a button so that none of the buttons functionality works

Im disabled button 

<beer-button disable id="disabled">Im a disabled button</beer-button>
