Using Text

To add some text with the beer-brand tag, simply add an id attribute and then set the inner html to whatever you like

Team Beer 

    <beer-brand  id="brand">Team Beer</beer-brand>

Using an Image

To add an image for your brand, simply add the brand image attribute with a path to the image from where the .html is located. The default height and width for an image are set to 30. 


    <beer-brand brandImage="./beer_taeam_logo.png"></beer-brand>

Alternate Image

You can use the alt attribute of an image to give the beer-brand tag a second image to load if their is a problem with the one you supply to brandImage using the imageAlt attribute


    <beer-brand  brandImage="./beer_team_logo.png"ImageAlt="../Raw-Button/images/test.jpg"></beer-brand>

Adjusting the Height and Width

You can adjust the height and width of the image you supply the beer-brand tag by using the brandHeight and brandWidth attributes, you can chose to add them both or simply one or the other. 


    <beer-brand  brandImage="./beer_team_logo.png"brandHeight="200"brandWidth"200"></beer-brand>

Adding Link Functionality

You can add link functionality to the beer-brand tag by using the brandHref property


    <beer-brand  brandHref="../Raw-Brand/raw-beer-nav.html"></beer-brand>

Adding CSS

You can add CSS to beer-brand tag in the same way you would normally do it by simply using the attributes beerClass, imageClass, and newStyle beerClass will change the anchor elements class of the Brand and imageClass will change the class of the image element of beer-brand


    <beer-brand 
    beerClass="navbar-brand"
    brandImage="./beer_team_logo.png"
    ImageClass="d-inline-block align-top"
    newStyle="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css"    
    ></beer-brand>
