Beer-Nav

Creating a navbar using beer-button

The nav bar uses beer-buttons and adding these buttons to the navbar is as simple as creating the button you want and then passing the navbar and the button the navBarID attribute. There are two ways in which this attribute can be used 


      <beer-button  navBarID="bill">Im a button</beer-button>
      <beer-navbar  navBarID="bill"></beer-navbar>

And


        <beer-button  navBarID>Im a button</beer-button>
        <beer-navbar  navBarID></beer-navbar>

The first will allow you to specify which navBar you want the button to go to. When adding Buttons to the navBar they will appear in order.
