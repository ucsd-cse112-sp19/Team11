Creating a navbar using beer-button

The nav bar uses beer-buttons and adding these buttons to the navbar is as simple as creating the button you want and then passing the navbar and the button the navBarID attribute. There are two ways in which this attribute can be used 


    <beer-button id="name">Im a named button</beer-button>
  

      <beer-button navBarID="bill"></beer-button>
      <beer-navbar navBarID="bill"></beer-navbar>

And


        <beer-button navBarID></beer-button>
        <beer-navbar navBarID></beer-navbar>

The first will allow you to specify which navBar you want the button to go to. When adding Buttons to the navBar they will appear in order.

Adding Style

Simply pass in any CSS file you want to the newStyle attribute then use the beerClass attribute to designate which class styles you want to use with the beer-navbar. 


      <beer-button navBarID></beer-button>
      
      <beer-navbar navBarID beerClass="navbar navbar-light bg-dark" newStyle"https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css"></beer-navbar>

Adding A Brand

adding a brand to beer-navbar is as simple as adding a button simply provide the navBarBrandID to both a beer-navbar and a beer-brand. Just as with beer-button you can pass a string to navBarBrandID to get it to link to a specific beer-navbar note: if you don't pass ID and inner html attribute to beerBrand then nothing will actually show but you can still right click and check the inspector to see that the beer-navbar is actually there 


      <beer-brand navBarBrandID id="beerbrand">Team11</beer-brand>

      <beer-navbar navBarBrandID ></beer-navbar>
    
      <beer-brand navBarBrandID"navNav" id="beerbrand">Team11</beer-brand>

      <beer-navbar navBarBrandID="navNav"></beer-navbar>
